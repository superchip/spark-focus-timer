Privacy & Data Use

Spark is designed to respect your time and your privacy. We do not collect, transmit, sell, or monetize personal data. All state and preference storage happens locally in Chrome's extension storage areas.

What We Store

  • Timer state & statistics (focus sessions completed, total focus minutes, streak). Stored locally via chrome.storage.local. Resettable by you.
  • Preferences (durations, content type toggles, notification preference). Stored via chrome.storage.sync so they can roam across Chrome profiles signed in with sync enabled.
  • Optional debug logs are only kept if you enable Debug Mode. These never leave your machine and can be cleared.

What We Do NOT Collect

  • No browsing history
  • No personally identifiable information
  • No analytics, telemetry, tracking pixels, or fingerprinting
  • No third-party advertising libraries

Network Requests

When break content types are enabled we call only the public APIs you opted into:

  • Facts: https://uselessfacts.jsph.pl/
  • Quotes: https://api.quotegarden.io/

Responses are rendered directly in a local tab; nothing is forwarded elsewhere.

Permissions Justification

  • storage: save your preferences, timer state, stats.
  • alarms: drive focus / break timers reliably in the background.
  • notifications: inform you when a session ends (can be disabled).
  • host_permissions: limited to the small set of public APIs needed for content.

Removal & Data Deletion

Uninstalling the extension removes all stored data. You can also clear everything from the in‑popup Debug Console (when Debug Mode is enabled) or via the Options page reset button.
